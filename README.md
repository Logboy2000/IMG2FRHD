# IMG2FRHD
An image to freerider track converter written in pure Javascript (for better or for worse)

## Usage

- Visit [logboy2000.github.io/IMG2FRHD/](https://logboy2000.github.io/IMG2FRHD/).
- Upload an image
- Tweak settings
- Press 'Generate'
- Copy the track code!
